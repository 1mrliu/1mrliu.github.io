使用Git上传本地代码到Github

1.进入本地的项目目录，右键“Git Bash here”,调出git命令行界面，然后输入
git init
2.就是将目录下的所有文件上传
git add .
如果想添加某个特定的文件，只需把.换成特定的文件名即可
3.将项目提交到gitHub
git commit -m “注释语句”
4.在github上创建新的repository
这个步骤比较简单，直接去Github上边去创建即可。
5.点击 “Create repository”跳转到一个连接，可以获取到创建的仓库的https或者ssh地址
6.将本地的代码关联到github上
git remote add origin 项目的github地址
（如果使用https链接的话，直接就可以链接。
如果使用ssh链接的话的需要配置ssh）
7.上传代码到github之前需要先pull
git pull origin master
8.上传代码到远程仓库
git push -u origin master
